Keyboard Shortcuts

  - Copia (seleção ou linha)
    - Ctrl + C
  
  - Recorta
    - Ctrl + X 

  - Cola
    - Ctrl + V

  - Desfaz 
    - Ctrl + Z 
  
  - Refaz 
    - Ctrl + Shift + Z 

  - Move a linha 
    - Alt + ↑   ou  Alt + ↓ 
  
  - Seleciona o próximo parecido
    - Ctrl + D 
  
  - Comenta o código 
    - Ctrl + / 
  
  - Esconde a Side Bar
    - Ctrl + B
