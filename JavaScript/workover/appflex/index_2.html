DOM

DOM (Document Object Model) ou Modelo de Documento de Objeto. É uma forma de se manipular o layout do site através do código. Ele também é conhecido como API (Application Programming Interface)

O DOM cria um modelo da página HTML.

O DOM é chamado de objeto porque o modelo que também é conhecido como "Árvore DOM" é feito de objetos.

Os objetos são as tags HTML. Cada objeto representa um nó

1º nó -> document: que é o próprio documento HTML

2º tipo de nó: são os nós de elemento, ou as tags

3º tipo de nó: é o atributo status que é parte do elemento img. Identifiquei esse elemento para modificar a imagem pelo JS.
