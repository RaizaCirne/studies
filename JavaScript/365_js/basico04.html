Básico 04: Manipulando string


Escreva uma função que recebe dois parâmetros (text,flag).

Se o flag for igual a "start", deve retornar os 3 primeiros caracteres do parâmetro text.

Se a flag for igual a "end", deve retornar os 3 últimos caracteres do parâmetro text.

Se o parâmetro text tiver menos de 3 caracteres, apenas retornar o text.

Parâmetro com valor default

Note que o parâmetro flag recebe um valor na própria função.

Isso é o valor padrão do parâmetro flag, caso nenhum valor seja passado.

E por isso, o último console.log, que não passa nenhum valor para flag, tem o mesmo resultado do primeiro. Pois a flag tem o valor "start" por padrão.
