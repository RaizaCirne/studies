Desafio JavaScript

Básico 01

Para quem está iniciando, a prática de exercícios e resolução de problemas é a melhor forma de fixar o conteúdo.

Então, vou postar uma série de desafios, em vários níveis de dificuldade, para auxiliar na parte prática.

E depois, será publicado no stories a resolução do problema.

Bora começar?

O primeiro desafio, é o seguinte:

Escreva uma função que recebe dois valores, a e b, como parâmetros. Essa função deve retornar true se os dois valores forem iguais e do mesmo tipo. Caso contrário, false.

Resultado esperado:

myFunction(2,3) // retorna false

myFunction(3,3) // retorna true

myFunction(1,"1") // retorna false

myFunction("10","10") // retorna true
